Welcome:
Welcome to the Direct+ Documentation! Direct+ is a programming language built in Scratch, created by SML_365. Direct+ is a powerful yet learnable language for creating modular and interactive programs using the run() system and control flow.
This documentation's modules are designed to be read left-to right, as shown:

Feel free to navigate through the documentation using the links above or by using the buttons on the homepage. So, without further ado, let's dive into the world of Direct+!
Using the Text Editor:
The Direct+ text editor is a simple, mostly keyboard-based interface for writing and editing Direct+ code. The editor provides features for syntax coloring and highlighting, and basic dynamic error reporting as you type. To get started, head over to Direct+ on Scratch or TurboWarp (Recommended).
After starting the project, you should be able to type in the text editor.
Keyboard Shortcuts:
Since Direct+ is built in Scratch, it can't detect every key. Here are the important keyboard shortcuts that you'll be using often:
In editor:
[Up Arrow] = Backspace (Scratch can't detect the Backspace key)
[Left / Right Arrows] = Move Cursor
[` + Arrow Keys] = Scroll (All Directions)
[` + s] = Save Current File
[` + l] = Load File
[` + f] = Open Filesystem
In filesystem:
[` + Arrow Keys] = Scroll (Up and Down Only)
[` + n] = New File
[` + d] = Delete File
[` + r] = Rename File
